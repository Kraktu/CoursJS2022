Je vais à travers ce fichier lister la démarche pour importer un fichier image, ça sera utile pour moi et pour tout le monde. Je vais systématiser le process pour un PSB dans un premier temps.

Body
...

A savoir
...

l'Order in Layer des Cartes se trouvent à partir de 1.000 :

ElementOrder in Layer MinOrder in Layer Max
Background10001100
Fond (texte, personnage, type,...)11001200
Type12001300
Texte12001300
Personnage12001300
Cadre Carte13001400
Décoration au dessus de la carte14002000

Prérequis
...

  • Avoir pull le repo sur github et être à jour
  • Ouvrir le projet sur Unity

Les Actions
...

  1. Importer le fichier dans le bon dossier dans la structure
  2. Glisser l'élément dans une scène de test pour voir ce que l'on fait

  1. Dans les fichier, seléctionner l'élément et changer dans l'inspecteur :
    1) Les pixels par unit pour avoir le rendu voulu
    2) Le pivot pour le mettre au centre

  2. Unpack le prefab dans la hierarchie pour faire les modification

  3. Faire les modifications qu'il faut pour que le rendu soit correct avec l'order in layer (voir tout en haut les valeurs par défaut !!) Attention à bien ordonner les layer avec des espaces de 10 entre chaque calques lorsqu'ils ne sont pas sur le même plans !!, ici je mets les yeux à 1210 alors que le reste est à 1200, comme ça, si plus tard on rajoute des éléments, on pourra les incorporer entre 1200 et 1210 !

  1. Une fois le fichier terminé, refaire un prefab de l'élément terminé, et archiver le fichier originel.
