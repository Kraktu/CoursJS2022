Rappel des formulaires HTML

Header :

Les formulaires HTML sont des éléments clés dans le développement web, permettant aux utilisateurs d'interagir avec les pages en saisissant des données. Ils sont utilisés pour collecter des informations telles que des textes, des sélections, des cases à cocher, etc. Un formulaire est généralement composé d'éléments tels que <input>, <select>, <textarea>, et d'autres.

Body

Structure de Base d'un Formulaire HTML

Un formulaire HTML est défini par la balise <form>. À l'intérieur de cette balise, vous placez les éléments de formulaire que vous souhaitez utiliser.

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Formulaire</title>
</head>
<body>

    <form action="traitement.php" method="post">
        <!-- Les éléments du formulaire vont ici -->
		<label for="nom"> Nom :</label> <input type="text" name="nom" id="nom"> 
		<label for="email"> E-mail :</label> <input type="email" name="email" id="email">
        <input type="submit" value="Envoyer">
    </form>

</body>
</html>

Dans cet exemple, le formulaire envoie les données vers un script PHP nommé "traitement.php" lorsqu'il est soumis. Les champs de texte (<input type="text">) et de courrier électronique (<input type="email">) sont utilisés pour collecter des informations.

Principaux Éléments de Formulaire

1. <input>

L'élément <input> est polyvalent et peut être utilisé pour différents types de saisies, tels que du texte, des mots de passe, des cases à cocher, etc.

<label for="prenom">Prénom :</label><input type="text" name="prenom" id="prenom">  
<label for="motDePasse">Prénom :</label><input type="password" name="motDePasse" id="motDePasse">  
<input type="checkbox" name="accepteConditions" id="accepteConditions">  
<label for="accepteConditions">J'accepte les conditions d'utilisation</label>

2. <select> et <option>

Les éléments <select> et <option> permettent de créer des menus déroulants.

<select name="pays" id="pays">
    <option value="fr">France</option>
    <option value="us">États-Unis</option>
    <option value="ca">Canada</option>
</select>

3. <textarea>

L'élément <textarea> est utilisé pour les zones de texte de plusieurs lignes.

<label for="commentaire">Commentaires :</label><textarea name="commentaire" id="commentaire"></textarea>

4. <button>

La balise <button> est utilisée pour créer des boutons dans les formulaires.

<button type="submit">Envoyer</button>
<button type="reset">Réinitialiser</button>

Attributs Importants

action

L'attribut action spécifie l'URL du script qui traitera les données du formulaire lorsqu'il est soumis.

method

L'attribut method indique la méthode HTTP à utiliser lors de l'envoi des données. Les valeurs courantes sont "post" et "get".

name

L'attribut name est utilisé pour identifier chaque champ lors de l'envoi des données.

Conclusion

Les formulaires HTML sont une composante essentielle de l'interaction utilisateur sur le web. Ils permettent de collecter des données qui peuvent être traitées côté serveur avec des langages comme PHP. Assurez-vous de comprendre les différents éléments et attributs pour créer des formulaires efficaces et conviviaux.
