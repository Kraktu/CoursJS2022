Variables et Types de Données en PHP

Header :

Les variables sont des éléments essentiels de la programmation en PHP (et dans la plupart des langages de programmation). Elles sont utilisées pour stocker des données telles que des nombres, des chaînes de caractères, des tableaux, des objets, etc. Dans ce chapitre, nous allons explorer les variables et les types de données de base en PHP.

Body

Déclaration de Variables

En PHP, vous pouvez déclarer une variable en utilisant le symbole $ suivi du nom de la variable. Voici comment vous pouvez déclarer une variable :

$nomDeLaVariable = valeur;

Exemple :

$age = 30; // Une variable nommée $age contenant la valeur 30
$nom = "Carmen"; // Une variable nommée $nom contenant la chaîne "Carmen"

Types de Données

PHP prend en charge différents types de données. Voici quelques-uns des types de données de base :

  1. Entiers (int) : Pour stocker des nombres entiers, par exemple : $age = 30;.

  2. Décimaux (float) : Pour stocker des nombres décimaux, par exemple : $prix = 19.99;.

  3. Chaînes de Caractères (string) : Pour stocker du texte, par exemple : $nom = "Anthony";.

  4. Booléens (bool) : Pour stocker des valeurs de vérité (true ou false), par exemple : $estVrai = true;.

  5. Tableaux (array) : Pour stocker une collection de données, par exemple : $nombres = array(1, 2, 3);.

Variables Dynamiques

En PHP, les variables sont flexibles et dynamiques. Vous pouvez modifier le type de données d'une variable à tout moment. Par exemple, vous pouvez déclarer une variable comme suit :

$variable = 42; // $variable est un entier

Et ensuite, vous pouvez la changer en une chaîne de caractères :

$variable = "Bonjour"; // $variable est maintenant une chaîne de caractères

Affichage de Variables

Pour afficher le contenu d'une variable en PHP, vous pouvez utiliser la fonction echo.

Exemple :

$prenom = "Karim";
echo $prenom;

Concaténation de Chaînes

Pour concaténer (joindre) des chaînes de caractères en PHP, vous pouvez utiliser le point (.) comme opérateur de concaténation.

Exemple :

$nom = "Batiste";
$age = 30;
$message = "Bonjour, je m'appelle " . $nom . " et j'ai " . $age . " ans.";

Récupération des Entrées Utilisateur en Ligne de Commande

Lorsque vous exécutez un script PHP en ligne de commande, vous pouvez récupérer des informations fournies par l'utilisateur à l'aide de la fonction fgets(STDIN). Cette fonction permet de lire ce que l'utilisateur tape au clavier et de stocker ces données dans une variable.

Exemple d'Utilisation de fgets(STDIN)

<?php
// Demander à l'utilisateur de saisir son prénom
echo "Veuillez entrer votre prénom : ";

// Utiliser fgets(STDIN) pour lire l'entrée de l'utilisateur
$prenom = fgets(STDIN);

// Supprimer les caractères de nouvelle ligne ("\n" ou "\r") de l'entrée
$prenom = trim($prenom);

// Afficher le prénom de l'utilisateur
echo "Bonjour, " . $prenom . " !";
?>

Dans cet exemple, nous utilisons echo pour demander à l'utilisateur de saisir son prénom. Ensuite, nous utilisons fgets(STDIN) pour lire ce que l'utilisateur a saisi. Cependant, fgets inclut généralement un caractère de nouvelle ligne (\n) à la fin de l'entrée, nous utilisons donc trim pour supprimer ce caractère.

Astuces Importantes

  1. fgets(STDIN) retourne une chaîne de caractères, vous devrez donc la stocker dans une variable.

  2. Assurez-vous de nettoyer les données entrées par l'utilisateur, car elles peuvent contenir des caractères inutiles tels que les caractères de nouvelle ligne.

  3. Vous pouvez utiliser des variables pour stocker les entrées utilisateur et les manipuler comme n'importe quelle autre donnée.

  4. fgets(STDIN) attend que l'utilisateur saisisse des données et appuie sur Entrée pour envoyer les données au script.

L'utilisation de fgets(STDIN) est essentielle pour interagir avec les utilisateurs lors de l'exécution de scripts PHP en ligne de commande. Elle permet de créer des interfaces interactives dans la console pour recueillir des informations, effectuer des opérations et afficher des résultats en fonction des entrées de l'utilisateur.

Conversion de Chaînes de Caractères en Autres Types de Données

Après avoir récupéré une chaîne de caractères à partir de fgets(STDIN), vous pouvez la convertir en d'autres types de données selon les besoins. Voici comment vous pouvez effectuer ces conversions :

1. Conversion en Entier (int)

Pour convertir une chaîne de caractères en entier, vous pouvez utiliser la fonction intval() :

<?php
// Demander à l'utilisateur de saisir un nombre entier
echo "Veuillez entrer un nombre entier : ";

// Utiliser fgets(STDIN) pour lire l'entrée de l'utilisateur
$entree = fgets(STDIN);

// Convertir la chaîne de caractères en entier
$nombreEntier = intval($entree);

// Afficher le résultat
echo "Le nombre entier que vous avez saisi est : " . $nombreEntier;
?>

2. Conversion en Nombre Décimal (float)

Pour convertir une chaîne de caractères en nombre décimal (point flottant), vous pouvez utiliser la fonction floatval() :

<?php
// Demander à l'utilisateur de saisir un nombre décimal
echo "Veuillez entrer un nombre décimal : ";

// Utiliser fgets(STDIN) pour lire l'entrée de l'utilisateur
$entree = fgets(STDIN);

// Convertir la chaîne de caractères en nombre décimal
$nombreDecimal = floatval($entree);

// Afficher le résultat
echo "Le nombre décimal que vous avez saisi est : " . $nombreDecimal;
?>

3. Conversion en Booléen (bool)

Pour convertir une chaîne de caractères en booléen (true ou false), vous pouvez utiliser des conditions :

<?php
// Demander à l'utilisateur de saisir 'true' ou 'false'
echo "Veuillez entrer 'true' ou 'false' : ";

// Utiliser fgets(STDIN) pour lire l'entrée de l'utilisateur
$entree = trim(fgets(STDIN));

// Convertir la chaîne de caractères en booléen
$estVrai = ($entree === 'true');

// Afficher le résultat
echo "La valeur booléenne que vous avez saisie est : " . ($estVrai ? 'true' : 'false');
?>

Ces exemples vous montrent comment récupérer des données en tant que chaînes de caractères, puis les convertir en d'autres types de données en utilisant les fonctions de conversion appropriées. Cela vous permet de manipuler et d'utiliser ces données selon les besoins de votre application.

Exercices :

Exercice 1 : Saisie d'un nom et affichage

Écrivez un script PHP qui demande à l'utilisateur de saisir son nom, stocke le nom dans une variable, puis affiche un message de salutation avec le nom de l'utilisateur.

Exercice 2 : Calcul de l'âge

Écrivez un script PHP qui demande à l'utilisateur de saisir son année de naissance, calcule son âge en soustrayant l'année de naissance de l'année actuelle, stocke l'âge dans une variable, puis affiche l'âge de l'utilisateur.

Exercice 3 : Conversion de température

Écrivez un script PHP qui demande à l'utilisateur de saisir une température en degrés Celsius, convertit cette température en degrés Fahrenheit en utilisant la formule de conversion, stocke la température en Fahrenheit dans une variable, puis affiche la température en Fahrenheit.

Exercice 4 : Manipulation de chaînes

Écrivez un script PHP qui déclare deux variables contenant des prénoms, par exemple : 

$prenom1 = "Alice";
$prenom2 = "Bob";

Ensuite, utilisez ces variables pour afficher un message qui salue les deux prénoms, par exemple :

Bonjour Alice et Bob !

Exercice 5 : Addition de deux nombres

Écrivez un script PHP qui demande à l'utilisateur de saisir deux nombres entiers, additionne ces deux nombres, stocke le résultat dans une variable, puis affiche le résultat de l'addition.
