Accueil - Algorithmie

Header :

Vous trouverez plus bas la structure du cours d'algorithmie en pseudo-code utile à apprendre tous les langages de programmation. Le but de ce cours est de l'utiliser en parallèle d'un autre cours de programmation pour comprendre la logique et la structure avant de rentrer dans un concept précis propre au langage que vous voudrez étudier.

Body:

A relire à chaque nouveau chapitre

  • Algo - Utilisation des Concepts Appris pour Résoudre des Problèmes

Cours Principal

Chapitre 1 : Les Bases du Pseudo-Code

  • Algo - Structure Générale du Pseudo-Code
  • Algo - Variables et Types de Données
  • Algo - Opérations Mathématiques de Base
  • Algo - Entrée et Sortie de Données

Chapitre 2 : Structures de Contrôle

  • Algo - Structures Conditionnelles (SI...ALORS...SINON)
  • Algo - Boucles (POUR, TANT QUE)
  • Algo - Instructions de Saut (SAUTER, CONTINUER)
  • Algo - Structure CASE (Choix Multiple)
  • Algo - La Boucle POUR CHAQUE

Chapitre 3 : Structures de Données

  • Algo - Variables Composées (Tableaux, Listes)
  • Algo - Structure de données associatives (Dictionnaires, Maps)
  • Algo - Variables Composées (Enregistrements, Structures)
  • Algo - Allocation Dynamique de Mémoire

Chapitre 4 : Sous-programmes (Fonctions et Procédures)

  • Algo - Définition et Appel de Fonctions
  • Algo - Paramètres et Retours de Fonctions
  • Algo - Portée des Variables
  • Algo - Réutilisation de Code

Chapitre 5 : Gestion d'Erreurs et Exceptions

  • Algo - Gestion d'Erreurs avec des Structures Conditionnelles
  • Algo - Utilisation des Exceptions

Chapitre 6 : Récursivité

  • Algo - Principe de la Récursivité
  • Algo - Cas de Base et Cas Récursifs
  • Algo - Exemples de Fonctions Récursives

Chapitre 7 : Complexité Algorithmique

  • Algo - Notion de Complexité Temporelle et Spatiale
  • Algo - Analyse de la Complexité
  • Algo - Notation Big O

Chapitre 8 : Algorithmes de Tri

  • Algo - Tri à Bulles
  • Algo - Tri par Sélection
  • Algo - Tri par Insertion
  • Algo - Tri Rapide (Quicksort)
  • Algo - Comparaison des Algorithmes de Tri

Ressources Utiles

  • Algo - Commenter le Code
  • Algo - Nommage des Variables et des Fonctions
  • Algo - Organisation du Code
  • Algo - Tests Unitaires
  • Algo - Schémas de raisonnement résolution d'exercice de programmation
