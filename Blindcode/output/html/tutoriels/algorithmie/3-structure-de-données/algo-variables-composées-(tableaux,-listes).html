Tableaux (Listes)

Header :

Les tableaux, également appelés listes, sont des structures de données fondamentales en programmation. Ils permettent de stocker une collection d'éléments du même type dans un ensemble ordonné. Les tableaux sont largement utilisés pour gérer des données structurées et effectuer des opérations sur ces données. Dans ce chapitre, nous explorerons les concepts fondamentaux des tableaux en pseudo-code.

Body

1. Définition d'un Tableau

Un tableau est une collection ordonnée d'éléments, où chaque élément est accessible par un index. En pseudo-code, vous pouvez déclarer un tableau de la manière suivante :

TABLEAU nomTableau : TYPE_ELEMENT[TAILLE]
  • nomTableau est le nom que vous donnez à votre tableau.
  • TYPE_ELEMENT est le type de données des éléments stockés dans le tableau (ENTIER, REEL, CHAINE, etc.).
  • TAILLE est la taille du tableau, c'est-à-dire le nombre d'éléments qu'il peut contenir.

Exemple de déclaration d'un tableau :

TABLEAU nombres : ENTIER[5]

Dans cet exemple, nous déclarons un tableau appelé "nombres" qui peut contenir 5 entiers.

2. Accès aux Éléments d'un Tableau

Les éléments d'un tableau sont accessibles via leur indice. L'indice est un nombre entier qui commence généralement à 0 pour le premier élément, 1 pour le deuxième, et ainsi de suite. Pour accéder à un élément d'un tableau, utilisez la syntaxe suivante :

nomTableau[indice]

Exemple d'accès aux éléments d'un tableau :

TABLEAU nombres : ENTIER[5]
nombres[0] <- 10
nombres[1] <- 20
nombres[2] <- 30

AFFICHER "Le premier élément est ", nombres[0]
AFFICHER "Le deuxième élément est ", nombres[1]

Dans cet exemple, nous attribuons des valeurs aux trois premiers éléments du tableau "nombres" et les affichons en utilisant leurs indices.

3. Parcours d'un Tableau

Les boucles "POUR" et "TANT QUE" sont couramment utilisées pour parcourir un tableau et effectuer des opérations sur ses éléments. Voici un exemple de parcours d'un tableau avec une boucle "POUR" :

TABLEAU nombres : ENTIER[5]
nombres[0] <- 10
nombres[1] <- 20
nombres[2] <- 30
nombres[3] <- 40
nombres[4] <- 50

POUR i DE 0 A 4 PAS 1
    AFFICHER "Élément ", i, " : ", nombres[i]
FIN POUR

Ce code parcourt le tableau "nombres" à l'aide d'une boucle "POUR" et affiche chaque élément avec son indice.

Les tableaux sont un outil essentiel pour gérer des collections de données et effectuer des opérations sur ces données de manière structurée. Ils sont utilisés dans de nombreux domaines de la programmation pour stocker des informations telles que des scores de jeux, des listes d'utilisateurs, des données de capteurs, etc. Comprendre comment déclarer, accéder et parcourir les tableaux est fondamental pour devenir un programmeur efficace.

4. Les listes

Lorsque l'on va parler de liste, vous pourrez entendre parler de :

  • Un objet qui peut stocker des données de types différents, mais je ne l'utilise pas dans cette optique car elle n'est pas vraiment réellement applicable à ce que j'utiliserai moi au cours. nous utiliserons, dans les listes et les tableaux, toujours des éléments de même type, c'est pourquoi, lorsque nous parlerons de liste, nous parlerons du point suivant...
  • Un tableau dynamique, c'est à dire un objet dont la taille peut varier à la volée, déclarée sans forcément une taille de base, et auquel on vient ajouter et supprimer des éléments au fur et à mesure du code sans que cela ne pose de problèmes spécifiques aux tableaux que nous verrons directement dans les différents langages de programmation. En général, les listes viennent en plus avec tout un set de fonction qui permettent de les trier, ordonner, renverser, croiser, etc... rapidement !
