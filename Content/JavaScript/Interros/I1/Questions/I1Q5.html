Interro 1 - Question 5 : High Score

Vous trouverez en dessous le début d'un script. Celui ci génère aléatoirement des pseudos et des points pour 4 joueurs. Pour le moment le programme affiche les 4 joueurs dans l'ordre de leur déclaration dans le code.

Vous pouvez résoudre l'exercice de deux manière :

Soit le classement apparaîtra directement trié (attention a bien conserver les noms dans le bon ordre aussi je peux le vérifier!). A ce moment là vous devrez bidouiller dans les lignes de codes déjà inscrites.

Soit vous conservez tout le début de ma résolution et vous affichez dans une deuxième boite de dialogue le classement trié. Dans ce cas, vous pouvez simplement coder en dessous de ce qui a déjà été écrit.

Pas de nombre d'instruction pour cet exercice, cela dépend trop de la méthode que vous choisirez, si vous n'arrivez pas à tout faire, ce n'est pas grave, essayez de faire ce que vous pouvez !
