Interro 1 - Question 3 : Savoir Renvoyer

Votre script sera simplement composé d'une fonction. Le rôle de cette fonction sera de prendre en entrée un nombre, que vous choisirez, qui représentera l'âge d'une personne ainsi qu'un autre nombre. Cette fonction aura en valeur de retour l'âge additionné avec le deuxième chiffre reçu en paramètre. Le programme principal lui aura simplement comme rôle d'appeler cette fonction et d'afficher un petit message dans une boite de dialogue qui dira "Dans X ans vous aurez X ans".

Petite contrainte : Le chiffre additionné à l'âge sera un chiffre aléatoire entre 1 et 10. Pour ce faire, vous pouvez utiliser cette fonction :
Math.floor(Math.random() * 11)
