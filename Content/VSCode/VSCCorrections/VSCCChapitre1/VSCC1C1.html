Chapitre 1 - Exercice 1 : Installer et Configurer Visual Studio Code

Vous pouvez comme dit dans l'énoncé simplement aller sur https://code.visualstudio.com/ où vous trouverez assez simplement le bouton Download For Windows. Si vous avez Mac, déroulez le menu qui se situe à côté pour y trouver votre bonheur.

Lancez l'installeur une fois celui-ci téléchargé. Acceptez les termes du contrat de licence et choisissez l'endroit où installer VSCode. Si vous n'avez pas de partitions spécifiques sur votre disque dur où une quelconque configuration spéciale et que vous souhaitez l'installer précisément pour une bonne raison à un autre endroit, je vous conseille dele laisser s'installer là où il est prévu. C'est plus simple pour le retrouver sur toutes les machines pour la suite du cours. Passez à l'étape d'après qui vous demande simplement le nom que prendra VSCode dans le menu démarrer, on peut passer à la suite de nouveau.

Ici nous tombons sur le menu important. Vous pouvez premièrement cocher le fait d'avoir un raccourcis sur le bureau. Si à l'avenir vous pensez ne plus en avoir besoin, il sera encore temps de le supprimer. Mais surtout, ce qui nous importe, c'est que vous cochiez bien les cases suivantes :

  • Ajouter l'action "Ouvrir avec Code" au menu contextuel de fichier de l'Explorateur Windows
  • Ajouter l'action "Ouvrir avec Code" au menu contextuel de répertoire de l'Explorateur Windows

Les deux autres sont déjà cochées. Normalement toutes les cases sont maintenant cochées, on peut passer à la suite ! Plus qu'à cliquer sur installer et c'est parti. Cliquons sur terminer pour lancer automatiquement VSCode et passer à l'étape suivante.

Si vous voyez, lors de ce premier lancement vous pouvez décider d'un thème de couleur. Si vos yeux demandes certains contrastes particuliers, saisissez cette occasion de configurer les couleurs, sinon, le thème de base est très reposant pour le code, je vous conseille vraiment d'éviter les thèmes clairs, encore une fois, sauf si votre DV le demande ! Mais sinon, passons à l'installation du français, pour ce faire, on peut faire le raccourcis ctrl + shift + x pour atteindre les extensions, sinon, visuellement, c'est le dernier icône de la barre en haut à gauche, qui ressemble à un petit carré divisé en 4 dont 1 des carrés est en train d'arriver. Une fois dans le menu en ayant choisi une des deux solutions pour y arriver, vous aurez une barre de recherche dans laquelle vous pouvez tapper "french". Vous pouvez alors installer les extensions suivantes :

  1. French Language Pack for Visual Studio Code qui vous permettra de mettre VSCode en Français
  2. French - Code Spell Checker qui lui permettra de corriger les fautes d'orthographes de ce que vous écrirez. Aucune obligation pour celui-ci, il ne gênera pas si vous l'avez, mais à titre personnel, je ne jugerai pas votre orthographe !

Ne reste plus qu'à redémarrer VSCode pour que ce que nous avons fait prenne effet. C'est exceptionnel, la plupart des extensions que nous installerons ne demanderons pas de redémarrage, mais ici comme nous changeons entièrement la langue du programme il faut le faire.

Après avoir redémarré, vous pouvez appuyer sur F1, et tapper "Show Spell Checker configuration info". Logiquement il vous le proposera après avoir simplement tapé les mots "show spell", à ce moment la touche enter devrait vous ouvrir le bon menu. Une fois dans le bon menu, vous pouvez voyager à l'aide de la touche tabulation jusqu'à trouver les langues que l'on veut cocher pour la correction orthographique, dans notre cas le Français. Ne décochez surtout pas l'anglais, nous allons beaucoup, beaucoup, beaucoup écrire en anglais aussi !

Enfin, il ne reste vraiment plus qu'à optimiser VSCode pour les lecteurs d'écran. Si vous n'avez pas de lecteurs d'écrans vous pouvez skipper cette partie, sinon il suffit en général de lancer VSCode avec votre technologie d'assistance, alors un pop-up vous proposera d'optimiser votre VSCode pour les lecteurs d'écrans. Si vous avez un soucis, vous pouvez m'appeler ! SI jamais vous voulez encore plus optimiser VSCode pour votre handicap, je vous invite à lire de la documentation en ligne sur les polices d'écritures, les thèmes de couleurs ou autre chose qui vous vient à l'esprit et on en discutera ensemble pour essayer d'installer ça chez vous.
