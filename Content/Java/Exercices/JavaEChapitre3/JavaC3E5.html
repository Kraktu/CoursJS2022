Chapitre 3 - Exercice 5 : Un avant goût de la Factorisation

Imaginez une fonction qui aurait 2 paramètres. 2 nombres entiers. Elle aurait 2 instructions, la première est d'afficher l'addition de ces 2 nombres à l'écran ainsi que le résultat de celle-ci, la deuxième et de faire la même chose avec la soustraction. Une fois lancé, votre programme devrait afficher quelque chose comme ceci :

x + y = A x + y = S

où x est le premier nombre passé en paramètre, y le deuxième, A le résultat de l'addition et S le résultat de la soustraction.

Une fois ceci fait, le vrai exercice commence. Commentez votre code car maintenant votre code devra comporter deux fonction. Chacune de ces fonctions fera une des deux opérations mathématique.

Enfin, essayez, à l'aide d'un seul appel de fonction dans le Main, d'appeler les deux fonctions qu'on a créé.

Indice 1

Pour déclarer un nombre entier, le mot clef n'est pas string mais int

Indice 2

Pour commenter du code, on sélectionne la partie de code à commenter et on fait le raccourcis ctrl + :

Indice 3

Il y a plusieurs façons de résoudre ce problème :

  • Appeler la fonction de la soustraction depuis la fonction de l'addition, et appeler l'addition dans le Main.
  • Créer une troisième fonction qui appellera les 2 autres fonctions.
