Chapitre 3 - Exercice 1 : Java 3 en 1, le produit préféré des ménagères !

Créez votre première fonction. Celle-ci aura comme utilité de faire les 4 choses qu'on a fait dans le chapitre 1 à chaque fois qu'elle est appelée ! Elle devra donc :

  1. Sur une ligne écrire Hello World
  2. Créer une variable, y stocker votre jeu préféré, et afficher une phrase comme "My favorite game is X."
  3. Demander à l'utilisateur son animal préféré et afficher : "Your favorite animal is X "
  4. Faire faire un Bip à l'ordinateur
Indice 1

Pour créer une fonction, vous pouvez taper :
static void X()
{
// Ici vous pourrez écrire votre code !
}
Bien sur, à la place du X, vous devez mettre le nom de votre fonction, un nom qui correspond le mieux à notre fonction.
Ceci doit être fait en dehors du Main !

Indice 2

Pour appeler une fonction, il suffit de mettre son nom, suivi de d'un couple de parenthèse. Les fonctions auront toujours besoins de leurs parenthèses, ne les oubliez pas !
