Chapitre 3 - Exercice 6 : Le futur simple

Créez une fonction, son rôle sera simplement de demander l'âge de la personne qui utilise le programme à l'aide d'un scanner. Ensuite, une fois la donnée récupérée, votre programme calculera une simple addition pour savoir dans 17ans, quel âge aura la personne. Renvoyez ce résulta au scope principal (le Main). Le rôle de votre Main sera, forcément, d'appeler cette fonction, mais aussi d'écrire ensuite une petite phrase qui dira :
Dans 17 ans vous aurez X ans
Où X sera le résultat de l'addition calculé dans notre fonction.

Indice 1

Pour renvoyer une information à ce qui appelle la fonction, il faut utiliser l'instruction return. ce qui suis le return sera renvoyé à ce qui appelle la fonction.

Indice 2

Pour récupérer une information renvoyée par une fonction, il y a deux solutions, soit vous l'utilisez directement. Donc vous pouvez utiliser votre fonction comme une variable, soit vous stockez la valeur de retour dans une variable, un peu comme vous avez fait avec le scanner.
