Chapitre 2 - Exercice 2 : A quoi tu joues ?

Déclarez une variable qui s'appelle game. Dedans est stocké le nom de votre jeu (vidéo ou autre si vous ne jouez pas) préféré. Essayez d'afficher dans votre console à l'aide de la fonction println :
My favorite game is X.
X étant votre jeu préféré.

Indice 1

Le type de variable pour une phrase, est la chaîne de caractère, en programmation on dit une string.

Indice 2

Pour concaténer (lier) des chaînes de caractères, qu'elles soient stockée dans une variable ou pas, on peut utiliser le signe +. Comme en mathématique + additionne deux chiffres, en programmation + additionne en fonction du type de donnée qui est additionné. L'addition de deux chaînes de caractère résulte en le fait que la deuxième suivra la première.
