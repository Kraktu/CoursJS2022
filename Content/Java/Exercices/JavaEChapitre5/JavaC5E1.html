Chapitre 5 - Exercice 1 : Première rencontre avec le while

Essayez d'écrire un bout de code qui va écrire 5 fois Hello World à l'aide d'une boucle while . Ensuite, posez vous la question de comment vous devriez modifier ce code si vous vouliez écrire 5000 fois Hello World. Si vous avez réussi, une seule ligne devrait être à modifier !

Indice 1

On peut voir le "while" comme un "if". Sauf qu'à la différence du if, le code se trouvant dans un while est exécuté en boucle le temps que l'expression booléenne passe à false.
