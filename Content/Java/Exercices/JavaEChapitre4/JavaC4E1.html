Chapitre 4 - Exercice 1 :Qu'elle heure est-il au Java dis ?

Votre programme devra demander à l'utilisateur l'heure qu'il est en 2 fois. La première fois vous demanderez seulement l'heure et la deuxième fois seulement les minutes. Stockez les 2 informations dans deux variables différentes. Après récolte de ces informations, le programme écrira l'heure qu'il est en entier. Si l'heure n'est pas correctement encodée (minutes supérieur à 59, heures supérieures à 24,...) alors le programme affichera plusieurs messages d'erreur du style "L'heure encodée est supérieur à 60", etc. pour faire comprendre les problèmes.

Après ceci, quoi qu'il arrive, le programme souhaitera une bonne journée à l'utilisateur.

Indice 1

Le déroulement du programme pourra ressembler à ceci :

Quelle heure est-il ? (hh)
user : 28
Quelle heure est-il ? (mm)
user : 75
Erreur : L'heure encodée est supérieure à 24
Erreur : Les minutes encodées sont supérieures à 60
Bonne journée

Indice 2

Pour regarder la validité d'une information, on peut utiliser ce qu'on appelle un if. Cela veut littéralement dire 'si' en anglais. La structure est donc comme suit :

if(condition)
{
 //code
}

La condition étant une opération booléenne ayant comme réponse possible soit vrai soit faux.
