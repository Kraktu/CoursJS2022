Chapitre 1 - Exercice 2 : Installer Java

On va maintenant essayer de tester si tout ce qu'on a fait avant marche !

  1. Allez dans le dossier ou vous stockez tous vos cours pour la formation, créez-y un dossier pour le cours de Java Procédural (par exemple JavaProc). Dedans, on va créer un ficher que l'on va appeler Hello.java (attention, Hello avec une majuscule !!)
  2. Ouvrez ce fichier avec VSCode comme appris au dernier cours (Ctrl + o)
  3. Pour le moment je ne rentrerai pas dans les détails, le but est de voir si tout ce qu'on a fait marche. Donc premièrement on va dans le fichier écrire :
    class et vous allez, à l'aide des flèches directionnelles, sélectionner le deuxième choix en appuyant sur enter après avoir appuyé une fois sur la flèche du bas.
  4. Un petit code s'affiche, pas d'inquiétude pour le moment, ce qui nous intéresse c'est de place notre curseur dans la class hello, juste en dessous de l'accolade ouvrante qui suis la parenthèse fermante, si vous avez suivi tout parfaitement, c'est la ligne 7 chez vous. Dedans, vous pourrez tapper :
    main et puis appuyer sur enter.
  5. Même chose, de l'autocomplétion se fait, et tout du code se met, on expliquera ça plus tard, mais nous avons enfin les choses minimale pour nous permettre de coder.
    En dessous de la ligne du main, nous pouvons écrire :
    sout
    puis appuyer sur enter. Dans la ligne qui apparaît, on peut se placer avec le curseur dans les parenthèses et taper :
    "Hello World"
    N'oubliez pas les guillemets !
  6. On peut enfin tout sauvegarder ! Il est l'heure d'exécuter tout ce code que nous venons de créer. Pour ce faire, on va ouvrir l'invite de commande, pour ça vous allez faire le raccourcis :
    windows + R
    Et tapper dans la barre de recherche qui est apparue :
    cmd
    et appuyer sur enter.
  7. Je ne rentrerai pas dans les détails non plus ici, c'est le but du cours de Johnny en partie, mais sachez simplement que vous avez face à vous votre ordinateur, mais sans interface graphique, c'est un petit peu comme les vieux ordinateur avant que Windows n'existe, les ordinateurs sous DOS !
    De base, vous vous trouvez dans le dossier :
    C:\Users\VotreNomDUtilisateur
    c'est à dire le dossier racine de votre session Windows. On peut donc aller chercher le dossier que vous avez créé en voyageant à l'aide de la commande CD. Par exemple, si je veux aller sur mon bureau en ligne de commande à partir de là, je tape :
    CD Desktop
    Une autre méthode est de tapper CD, suivi d'un espace, et puis faire des tabulations pour voir tous les fichiers et dossiers présents dans le répertoire où je suis jusqu'à trouver celui qui m'intéresse, dans notre cas Desktop.
  8. Répétez cette méthode pour entrer dans le dossier dans lequel vous avez créé votre dossier Hello.java !
  9. Lorsque vous êtes dans le bon dossier vous pouvez exécuter la ligne suivante :
    java Hello.java
    Ou comme tout à l'heure, tapper java, suivi d'un espace, suivi de tabulations jusqu'à trouver le bon fichier à exécuter. Si tout s'est bien passé, votre invite de commande devrait vous dire :
    Hello World
    Si ça ne marche pas, appelez moi, car c'est qu'il y a un soucis !
