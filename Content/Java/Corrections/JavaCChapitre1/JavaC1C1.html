Chapitre 1 - Exercice 1 : Installer Java

Pour le cours de Java nous allons avoir besoin de 3 choses différentes, Java, Visual Studio Code et l'invite de commande que l'on appellera CMD, nous avons déjà vu ensemble comment marche Visual Studio Code avec notre petit exercice d'HTML, il ne nous reste donc plus qu'à nous occuper de Java et de CMD.

  1. nous allons nous rendre à cette adresse :
    https://jdk.java.net/18/
    Pour télécharger et installer le JDK (Java Development Kit). Pour l'installation, il n'y a vraiment rien de compliqué, vous cliquez tout le temps sur suivant jusqu'à la finalisation de l'installation.
  2. Nous allons maintenant finaliser la liaison entre Java et Visual Studio Code, pour ce faire, dans VSCode, faites le raccourcis Control + Shift + X pour aller installer une extension, et dans la barre de recherche tapez Java, et installez la première extension, Extension Pack for Java.
