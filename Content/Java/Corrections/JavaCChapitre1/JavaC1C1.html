Chapitre 1 - Exercice 1 : Installer Java

Pour le cours de Java nous allons avoir besoin de 3 choses différentes, Java, Visual Studio Code et l'invite de commande que l'on appellera CMD, nous avons déjà vu ensemble comment marche Visual Studio Code avec notre petit exercice d'HTML, il ne nous reste donc plus qu'à nous occuper de Java et de CMD.

  1. nous allons nous rendre à cette adresse :
    https://jdk.java.net/18/
    Pour télécharger l'open JDK (Java Development Kit libre de droits).
  2. Pour l'installer, nous devons d'abord extraire le zip dans un dossier, pour ce faire, créez un dossier à la racine de votre dossier programs. Donc allez dans C: puis dans programs, nouveau dossier et renommez le Java. Dedans, extrayez l'archive trouvée sur le site du dessus.
  3. Vous pouvez supprimer l'archive une fois ceci fait, elle ne nous sert plus à rien !
  4. Il ne suffit plus maintenant que de montrer à Windows que nous avons Java et qu'il peut donc l'utiliser dans l'invite de commande. Pour ça, il faut l'ajouter au variables d'environnement de système. Un peu effrayant comme démarche lorsque l'on fait ça pour la première fois, mais pas d'inquiétude, ça va bien se passer. Appuyez sur votre touche Windows et tapez "variables d'environnement", il devrait vous proposer de les modifier, lancez cette fenêtre !
  5. Ici, faites très attention à bien suivre ces étapes de manière minutieuse. Premièrement cliquez sur le bouton "Variables d’environnements..." situé en bas de l'onglet "paramètres système avancés" sur lequel vous êtes normalement tombé. Vous pouvez maintenant, dans le tableau du dessus, c'est à dire les variables de votre utilisateur, aller mettre votre curseur sur "PATH" et cliquer sur modifier. Le raccourcis clavier lorsque vous avez votre curseur sur "PATH" est la touche M, sinon cliquez sur le bouton situé en dessous à l'aide de tabulation. Une fois rentré dans le Path, vous pouvez maintenant à l'aide des boutons ou de votre touche tabulation ou encore le raccourci clavier sur le touche N, cliquer sur le bouton nouveau. Là, vous pourrez y coller le chemin vers le dossier Java que l'on vient de créer. Normalement, ça devrait être : C:\Program Files\Java si vous n'avez pas fait autre chose que moi.
  6. Il est temps de voir si tout ça marche, cliquez sur OK pour quitter toutes les fenêtres que nous avons ouvert, et fermez tout ce qu'on a utilisé jusque maintenant, redémarrez votre CMD si elle est lancée.
  7. A parti de là, ce qui nous intéresse, c'est l'invite de commande. Je vais vous faire l'affront une unique fois de vous rappeler comment on le fait : appuyez sur votre touche windows, tapez CMD puis la touche enter !
  8. L'heure de vérité à sonné, si vous tapez la commande : java -version, il devrait être écrit quelque chose comme :
  9. openjdk version "18.0.1.1" 2022-04-22
    OpenJDK Runtime Environment (build 18.0.1.1+2-6)
    OpenJDK 64-Bit Server VM (build 18.0.1.1+2-6, mixed mode, sharing)
  10. Nous allons maintenant finaliser la liaison entre Java et Visual Studio Code, pour ce faire, dans VSCode, faites le raccourcis Control + Shift + X pour aller installer une extension, et dans la barre de recherche tapez Java, et installez la première extension, Extension Pack for Java.
