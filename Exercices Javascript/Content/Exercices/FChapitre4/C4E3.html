Chapitre 4 - Exercice 3 : Stopper une boucle

Cette fois-ci, toujours à l'aide d'un while, écrivez un programme qui demandera à l'utilisateur de tapper sur une touche Si la personne tape y, le programme s'arrête en disant "vous avez arrêté la boucle prématurément" Sinon, le programme continue à poser la question jusque 10x maximum. Après quoi s'arrête sans rien dire.

Le petit mot "break" permet de sortir d'une boucle quand on le veut.

Résolvez cet exercice une fois avec le mot break et une fois sans utiliser le mot break ! Le programme doit avoir EXACTEMENT le même état dans les deux cas. (toutes les valeurs doivent être les mêmes, le résultat le même, etc...)
